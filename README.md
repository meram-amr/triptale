# TripTales ✈️ — Personal Travel Blog

A fully responsive personal blog homepage built with React.js and Bootstrap 5.

## Features
- 🧭 Responsive Navbar with mobile hamburger menu
- 🌍 Hero Section with travel imagery
- 🔍 Real-time keyword search
- 🏷️ Category filter (Travel, Food, Culture, Tech, Adventure)
- 🃏 Blog post cards with image, title, description & date
- 📄 Pagination (6 posts per page)
- 👤 About Me section with stats & social links
- 📬 Contact form

## Built With
- React.js
- Bootstrap 5
- CSS Variables
- useState Hook
